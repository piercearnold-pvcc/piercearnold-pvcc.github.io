All about Github

  • Q. what is a version control system?
  • A. It is how Github tracks updates and changes to a given software

  • Q. What is Git?
  • A. A version control system

  • Q. What is Github?
  • A. A web based interface that uses git

  • Q. What is the difference between Git and Github?
  • A. Git is a version control system that allows developers to track changes in their code. GitHub is a web-based hosting service for git repositories

  • Q. Who started Github and how was it started?
  • A. Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chacon started it using Runy on Rails

  • Q. What company owns it now?
  • A. Microsoft

  • Q. How much does a Github account cost?
  • A. anywhere from free to $21 per month

  • Q. What is the Octocat?
  • A. Github's mascott, a half cat half octopus
