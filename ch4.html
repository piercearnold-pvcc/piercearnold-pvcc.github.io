Internet four-layer software
  • application layer
  • transprot layer
  • netowrk layer
  • link layer
  • The application layer consists of those software units such as clients and servers that use Internet communication to carry out their tasks.
  • The application layer uses the transport layer to send and receive messages over the Internet in much the same way that you would use a shipping company to send and receive packages.
  • It is the network layer's job to decide in which direction a packet should be sent at each step along the packet's path through the Internet.
  • The link layer has the responsibility of transferring the packet.
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
  • a digital network that transmits data by breaking it into packets and sending them through a series of switches
  • able to deal with errors and mistakes
  • falls back on other pathways for delivery when congestion or failure occur
